# Utilisation de Rspack

<Callout type="info">
  Cette page n'est pas encore traduite en français. En attendant, veuillez consulter la [version anglaise](/en-US/docs/frontend-modules/using-rspack) du guide.
</Callout>

Les apps frontend OpenMRS sont construites avec [Rspack](https://rspack.dev) en utilisant une configuration standard fournie par `openmrs-esm-core`. Rspack accepte le format de configuration de Webpack ; les rares modules encore sur Webpack utilisent la même configuration par défaut via `openmrs/default-webpack-config`.

Pour les détails complets (Module Federation, dépendances partagées, recettes de personnalisation, exception du module Angular `esm-form-entry-app`), reportez-vous à la [version anglaise](/en-US/docs/frontend-modules/using-rspack).
